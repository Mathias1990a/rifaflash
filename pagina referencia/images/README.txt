# Imágenes para la Landing Page

## Instrucciones para agregar imágenes reales:

### 1. Cámaras Dahua Originales
Descargar desde el sitio oficial de Dahua:
- https://www.dahuasecurity.com/products

Modelos a buscar:
- **Dahua Bullet 2MP**: HAC-HFW1200R-Z-IRE6
- **Dahua Bullet 4K**: HAC-HFW1801T-Z-A  
- **Dahua Dome 4MP**: HDBW2431E-S-S2
- **Dahua PTZ 25x**: SD49225T-HN
- **Dahua WiFi**: IPC-HDBW1320E-W
- **Dahua ColorVu**: HAC-HFW1409TLM-A-LED

### 2. Cercos Eléctricos Reales
Buscar imágenes de:
- Cercos eléctricos residenciales instalados
- Postes de cerco con alambres
- Energizadores de cerco eléctrico
- Instalaciones completas de perímetro

### 3. Nombres de archivos sugeridos:
```
images/
├── dahua-bullet-2mp.jpg
├── dahua-bullet-4k.jpg
├── dahua-dome-4mp.jpg
├── dahua-ptz-25x.jpg
├── dahua-wifi-3mp.jpg
├── dahua-colorvu-4mp.jpg
├── cerco-electrico-1.jpg
├── cerco-electrico-2.jpg
├── cerco-instalacion-1.jpg
├── cerco-instalacion-2.jpg
├── hero-camaras.jpg
├── hero-cerco.jpg
└── hero-instalacion.jpg
```

### 4. Reemplazar en el código:
Una vez tengas las imágenes, reemplaza las URLs de Unsplash por:
```html
<img src="images/dahua-bullet-2mp.jpg" alt="Cámara Dahua Bullet 2MP">
```

## Nota:
Las imágenes actuales usan placeholders de Unsplash.
Para una página profesional, se recomienda usar:
1. Fotos reales de tus instalaciones
2. Imágenes oficiales de Dahua (con permiso)
3. Fotos de stock compradas o libres de derechos
